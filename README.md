🌸 Flower Shop Management System 🌸

📌 Description
I built this application to manage orders, customers, and inventory in a simple desktop system. It helped me improve my understanding of database integration, Windows Forms and CRUD operations.

---

🛠 Technologies Used

* C# (.NET Framework)
* Microsoft SQL Server
* Windows Forms
* Visual Studio

✨ Features

* Customer management
* Product & inventory management
* Order processing system
* Basic billing functionality
* Easy data handling with SQL database

🗄 Database Setup

1. Open SQL Server Management Studio
2. Create a new database (e.g., `FlowerShopDB`)
3. Open the provided `FlowerShopDB.sql` file
4. Execute the script to generate tables

▶️ How to Run the Project

1. Open the `.sln` file in Visual Studio
2. Update the database connection string if needed
3. Build and run the application

💡 What I Learned
- Connecting C# applications with SQL Server
- Designing and managing relational databases
- Implementing CRUD operations
- Building a Windows Forms UI

🚀 Future Improvements

* Add user authentication system
* Improve UI/UX design
* Add reporting and analytics features
